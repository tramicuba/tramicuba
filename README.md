# TramiCuba

**TramiCuba** es una aplicación web progresiva (PWA) diseñada para facilitar la gestión de trámites legales, fiscales y administrativos en Cuba. Ofrece una interfaz moderna, accesible y responsiva, con funcionalidades que permiten seleccionar servicios, contactar por WhatsApp, visualizar horarios en tiempo real y realizar solicitudes de forma eficiente.

---

## ?? Características

- ? Instalación como app (PWA)
- ?? Interfaz adaptable a móviles y escritorio
- ?? Selección de servicios con validación dinámica
- ?? Contacto directo por WhatsApp, correo y ubicación
- ? Reloj animado en tiempo real
- ?? Notificaciones tipo Toastify
- ?? Modales interactivos para confirmación y contacto
